# Шпаргалка — Нейрограмотность

Интерактивная шпаргалка по AI и разработке в стиле Pip-Boy из Fallout 4.
Охватывает 9 категорий: AI-модели, инструменты разработки, DevOps, сети/API, генерация медиа, Git/CI-CD, тестирование, агенты — с иллюстрациями и определениями для каждого термина.

Стек: HTML + CSS + Vanilla JS, nginx, Docker.
Порт: `8080`.

---

## Запуск

```bash
docker compose up -d --build
```

Открыть в браузере: [http://localhost:8080](http://localhost:8080)

## Остановка

```bash
docker compose down
```

## Пересборка (после изменений)

```bash
docker compose up -d --build
```
